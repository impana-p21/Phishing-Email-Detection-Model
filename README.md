# Phishing Email Detection Model

## Description
A machine learning model that detects phishing emails using Scikit-learn by analyzing email text and URLs.

## Features
- Train on phishing and legitimate emails
- Extract text and URL features
- Predict whether an email is Phishing or Safe
- Display model accuracy
- Display confusion matrix
- Simple Flask web interface

## Technologies Used
- Python
- Flask
- Scikit-learn
- Pandas
- HTML
- CSS

## Future Enhancements
- Deep Learning
